# SettingsPlus

### Description
A TamperMonkey script for Cookie Clicker that adds QOL options, a cheat menu and a CPS counter GUI.

## Keybinds
"F2" - Toggle CPS GUI

"F4" - Toggle Cheat Menu

"F6" - Toggle Auto Clicker
